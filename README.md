# VS Code Context Menu Fix (System Installer & User Installer)

This repository provides guides to fix the **missing "Open with Code" option in the Windows context menu** for Visual Studio Code.

Sometimes after installing VS Code, the **right-click context menu integration does not appear or stops working**. The fix depends on the type of installation used.

## Files Included

This repository contains two Markdown guides:

* **system-installer-fix.md**
  A step-by-step guide for fixing the context menu when **VS Code is installed using the System Installer** (installed for all users on the machine).

* **user-installer-fix.md**
  A guide for fixing the context menu when **VS Code is installed using the User Installer** (installed only for the current user).

## Why This Exists

Many developers run into issues where:

* The **"Open with Code"** option is missing in the right-click menu.
* VS Code does not appear in the folder context menu.
* Windows Explorer integration breaks after installation or updates.

These guides explain **how to restore the VS Code context menu integration correctly depending on the installer type**.

## When To Use Which Guide

* Use the **System Installer guide** if VS Code was installed for **all users on the system**.
* Use the **User Installer guide** if VS Code was installed **only for your Windows account**.

## Goal

The goal of this repository is to provide a **simple and clear reference** for developers who want to quickly restore the **VS Code right-click "Open with Code" functionality** in Windows.
